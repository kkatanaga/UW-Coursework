Unzip DogFight.zip in the src folder
Make sure the package name is "DogFight"

Run DogFight.java as an application to start the singleplayer game
Refer to "readme - server.txt" to run our multiplayer version

Controls are:
q - turn left
e - turn right

w - pitch forward
s - pitch backward

a - roll left
d - roll right

shift - accelerate
ctrl - decelerate

space - shoot
